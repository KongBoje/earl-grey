      Editor support

      Editor support is a bit scattered at the moment. Please tell me if you are using these modules and file any issues you encounter.

      • Emacs: earl-grey-mode
      • Atom: language-earl-grey
      • CodeMirror: earlmode.eg; browserify+earlify it and include the file, it should work. Tell me if you try to work with it.

      If you wish to contribute editor support for an editor which is not yet supported, please consult these guidelines.
